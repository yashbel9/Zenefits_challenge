The following Search App contains two file:
1. index.html 
2. maps.css

index.html contains the code where the call the Google Places API to get all the markers and mark them on the map. 
maps.css contains the stylesheet used to style the Search App. 

To run the application: 
Step 1: Make sure index.html and maps.css are in the same folder. 
Step 2: Open index.html in the preferred browser. 
Step 3: Type in the search box any places you want to search and markers will be shown after searching for the places on the map.
